# Advanced Number Guessing Challenge

## Description

A console-based C++ game where players try to guess a randomly generated number within limited attempts.

## Features

- Multiple Difficulty Levels
- Random Number Generation
- Hot / Warm / Cold Hint System
- Odd / Even Hint
- Score Calculation
- Best Score Tracking
- Guess History
- Input Validation
- Statistics Dashboard
- Replay Option

## Technologies Used

- C++
- Standard Template Library (STL)
