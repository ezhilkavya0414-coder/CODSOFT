# Advanced Calculator

## Description
This is an advanced console-based calculator developed in C++ as part of the CODSOFT C++ Programming Internship.

The calculator performs basic arithmetic operations along with several advanced mathematical functions through a simple menu-driven interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power Calculation
- Square Root
- Percentage Calculator
- Calculation History
- Input Validation
- Menu Driven Interface
- Continue Calculations Option

## Technologies Used

- C++
- Standard Template Library (STL)

## How to Run

1. Compile the program.
2. Run the executable.
3. Choose the desired operation.
4. Enter the required numbers.
5. View the result.
